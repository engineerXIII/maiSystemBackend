# Информационная высоконагруженная система
```
Структура репозитория:
├ cmd - папка с главными файлами
├ config - папка с описанием и обработкой конфигурации приложения
├ db
│ └ migrations - скрипты миграции для БД сервиса
├ docker - файлы для локальной отладки с помощью Docker Compose
├ docs - Документация Swagger сервиса
├ internal
│ ├ middlewares
│ └ server - модуль http сервера
└ pkg - набор пакетов с некоторыми функциями для сервиса (логирования, мониторинг, ошибки...)
```
